# build
```shell
git clone https://github.com/AFKL-CUIT/fumofinger.git
cd fumofinger
go build .
```

# example
```shell
./fumofinger -t "http://bilibili.com"
./fumofinger -r "./test/assets/target.txt"
./fumofinger --fumo "没人看小石"
```
